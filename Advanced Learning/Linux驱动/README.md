# Linux 驱动开发学习

## 简介

本项目是学习 Linux 设备驱动开发的完整记录，基于 ARM9 S3C2440 开发板，通过理论笔记与实践项目相结合的方式，系统学习 Linux 驱动开发。

**硬件平台**: ARM9 S3C2440 (Samsung)
**参考来源**: 白球王的Linux设备驱动程序入门

## S3C2440 关键外设

- **GPIO**: 通用输入输出
- **UART**: 串口通信
- **Timer**: 定时器
- **中断控制器**: VIC (Vector Interrupt Controller)
- **内存**: SDRAM / NAND Flash

## 常用命令

```bash
# 编译模块
make

# 加载模块
insmod xxx.ko

# 查看模块
lsmod

# 卸载模块
rmmod xxx

# 查看内核日志
dmesg

# 创建设备节点
mknod /dev/xxx c 主设备号 次设备号
```

## 学习建议

1. **循序渐进**: 从 ARM 基础开始，理解硬件再学驱动
2. **理论+实践**: 每学完一章，动手完成对应实践项目
3. **阅读源码**: 多阅读内核源码和经典驱动代码
4. **调试技巧**: 学会使用 printk、KGDB 等调试方法
