# 个人学习记录仓库 

## 简介
本仓库用于记录我的编程学习之路，涵盖 C++、Python、Linux 应用开发、Qt 框架等技术的系统学习过程。通过代码实践、项目开发和笔记整理，持续提升编程能力。

##  学习模块

### 1. [C++ 学习](basic-learning/CPP/README.md)
- **进度**: day01 - day10 系统学习
- **内容**: 基础语法 → 面向对象 → 模板
- **重点**: 类与对象、继承多态、运算符重载

### 2. [Linux 应用开发](basic-learning/Linux应用/README.md)
- **进度**: 10 个阶段 + 2 个综合项目
- **内容**: Linux 基础 → C 语言 → 系统编程 → 网络编程
- **重点**: 进程线程、IPC 通信、Socket 编程

### 3. [Python 学习](basic-learning/PYTHON/README.md)
- **进度**: 入门 → 进阶 → 科研 → 面试 → 算法
- **内容**: 基础语法、GUI、数据分析、爬虫、CV
- **重点**: PyQt、OpenCV、LeetCode 刷题

### 4. [Qt 框架学习](basic-learning/QT/README.md)
- **进度**: day01 - day09 + 15 个综合项目
- **内容**: GUI 开发 → 多线程 → 网络 → 数据库
- **重点**: 信号槽、自定义控件、综合项目实战
