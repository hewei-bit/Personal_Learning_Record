# Qt 学习之路

## 简介
本文件夹记录了我学习 Qt 框架的完整过程，从基础组件到综合项目实战，涵盖 GUI 开发、多媒体应用、串口通信、数据库操作等核心内容。通过每日学习和项目实践，掌握 Qt 跨平台应用开发技能。

## 核心知识点
- **Qt 基础**: 信号与槽、布局管理、事件处理
- **GUI 组件**: 常用控件、对话框、菜单工具栏
- **绘图系统**: QPainter、自定义控件
- **多线程**: QThread、线程安全
- **网络编程**: TCP/UDP、HTTP
- **数据库**: SQLite、Qt SQL 模块
- **多媒体**: 音视频播放、摄像头捕获
- **串口通信**: 串口数据收发

## 开发环境
- Qt 版本: Qt 5.x
- 编译器: MinGW / GCC
- 开发工具: Qt Creator
